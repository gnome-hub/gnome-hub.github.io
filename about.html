We are creating a Gnome extension which can be installed with ease by a single command. The extension will serve as a hub for the user's notifications, widgets, apps, and system statistics.
Currently the extension is in the "Minimum Viable Product" phase where the user is able to see the 10 most recent notifications, a small weather widget, and live updates for the CPU and MEM usage system stats.
In future versions, we hope to be able to provide a customizable and intuitive UI, where the user is then able to add/remove widgets and other features, as well as add the extension to the Gnome store.
