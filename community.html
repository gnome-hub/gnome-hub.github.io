Team Updates
Progress as of 03/03/2022:
~ See gallery for pictures of progress ~
1. We have a easy to install gnome extension
   - Currently, it can be installed by cloning our repository and running the ./install.sh command
2. The extension displays the 10 most recent notifications and updates in realtime
3. We have live updates for CPU and MEM usage (system stats)
4. We have a API request generating the weather information based on a latitude and longitude input given to it (only needs to be setup once)

As for unresolved issues and next improvements:

1. Lack of a customisable, intuitive UI
   -We need to work on a way to let users change settings for the extension like adding/removing widgets. Currently, this can be done by commenting out sections of our code which isn't very intuitive
   -We need to make the notifications look more intuitive based on notification type etc instead of just having it display as text
2. Adding to the gnome extension store
   -Currently, the extension can be installed by cloning the repository and running a install.sh command however, in order to make it available to more people, we want to have it added to the gnome extension store
3. Updates to Websites
   -We are yet to add documentation to the website on updates & steps to install
